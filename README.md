# Gaming Username Generator 🎮

My third Python project.

This program asks the user for their name, favorite game, age, and favorite color, then generates eight creative gaming username ideas.

## Features

* User input
* Generates 8 gaming username ideas
* Displays the length of each input
* Shows the data type of the user's name
* Console output

## Example

Input:

* Name: Manwel
* Favorite Game: Minecraft
* Age: 14
* Favorite Color: Blue

Output:

* Manwel Minecraft
* Manwel 14
* Manwel Blue
* Minecraft Blue
* Minecraft 14
* Blue Manwel
* Manwel Gamer
* Blue Minecraft
